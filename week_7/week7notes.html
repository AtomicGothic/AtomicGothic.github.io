Week 7 Notes



IIFE stands for "Immediately Invoked Function Expressions." they are anonymous functions that are invoked as soon as they're defined.
Using callback functions is necessary for writing asynchronous functions. This is important if you want to call an API and not have your
user think that your program crashed.
When doing functional programming, make sure that you are not modifying the passed in values. You can return new data, but not at the cost of
modifying incoming data.
AJAX was invented to allow websites to dynamically get information from the server, which allows the web-app to have more features and performances
The jQuery library is an option rather than using the "wild frontier" of the Fetch API standard.
